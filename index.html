FAQ
Common questions
Is this really halal-compliant?
Yes. We exclude defense, alcohol, tobacco, fast food, and pharmaceutical companies. All strategies use spot trading only — no margin, no short selling, no derivatives. Forex pairs are only suitable for Islamic (swap-free) accounts. We recommend verifying with your local scholar.
Do I need a Twelvedata API key?
Yes, you'll need a free Twelvedata API key (takes 2 minutes to get at twelvedata.com). The free plan gives 800 calls/day which is enough for regular use. The scanner fetches real-time market data through this key.
Can I use this on my phone?
Yes. MuslimTrade is a Progressive Web App (PWA). After logging in, you'll see a prompt to "Add to home screen." Once installed, it works like a native app on both iOS and Android — no App Store required.
What is the access code?
The access code is included with your MuslimTrade PRO course purchase. Each code is unique and valid for lifetime access. You can use it on multiple devices.
What is the expected win rate?
The strategy targets a 73% win rate based on backtesting with a 2:1 risk/reward ratio (7% stop loss, 10%+ take profit). Past performance does not guarantee future results. This is a decision-support tool — you make the final call on every trade.
Does it work for intraday trading?
Yes. Timeframes include 1M, 5M, 15M, 30M, 1H, 4H, Daily, and Weekly. For intraday timeframes, the indicators use optimized parameters (EMA 5/13, RSI 8, ADX 10) to reduce noise and improve signal quality.
